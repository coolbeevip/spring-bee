## 编译和运行代码

* 编译代码
   ```bash
      $ mvn clean install -DskipTests
   ```

* 编译代码、运行单元测试
   ```bash
      $ mvn clean install
   ```

* 编译代码、检查代码风格并且运行相关的单元测试. 
  ```bash
     $ mvn clean install checkstyle:check
  ```
